# Pain Points Images

Esta pasta contém as imagens ilustrativas usadas no componente `PainPoints.tsx`.

## Estrutura de arquivos

As imagens devem ser nomeadas seguindo o padrão:
- `informacoes-espalhadas.jpg` ou `.png`
- `processos-complexos.jpg` ou `.png`
- `dependencia-operacional.jpg` ou `.png`
- `baixa-visibilidade.jpg` ou `.png`
- `dados-desarticulados.jpg` ou `.png`
- `escalabilidade-limitada.jpg` ou `.png`

## Formato recomendado

- **Formato**: JPG ou PNG
- **Dimensões**: Mínimo 800x600px (recomendado 1200x900px ou maior)
- **Orientação**: Horizontal (landscape)
- **Estilo**: Ilustrações isométricas 3D com tema verde/sustentabilidade

## Uso

As imagens são importadas através do arquivo `painPointsImages.ts` localizado em `src/assets/images/`.
